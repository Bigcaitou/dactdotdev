      站点搭建

      本站点的搭建通过如下工具实现：

      • 静态网站文档生成器： docsify@4.12.1
      • 代码高亮： prismjs
      • 代码及文档托管： github
      • 域名指向
        • www.dact.dev = bigcaitou.github.io → ...(github page接入点)
      • 静态文件托管
        • 通过 github page 实现
        • 通过github接入层自带的重定向功能
          • 将 https://www.dact.dev 重定向到 → 对应的repo https://www.github.com/Bigcaitou/dacaitou → 对应的 github page 的 public 目录
        • 配置方式详见docsify-部署-GitHub Pages
      • 评论系统：gitalk
